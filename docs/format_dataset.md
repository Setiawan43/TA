# Format Dataset

## CSV Harga Saham
Kolom wajib:
```text
date,close
```

Kolom tambahan yang diperbolehkan:
```text
open,high,low,volume
```

## CSV Laporan Keuangan
Kolom wajib:
```text
year,net_income,total_equity,total_assets,total_liabilities,shares_outstanding,market_price
```

## Catatan Satuan
Gunakan satuan rupiah penuh agar hasil EPS, BVPS, dan nilai intrinsik konsisten.
